
# AI科技日报-2026-01-06 SeedFold超越AlphaFold3
### **AI内容摘要**
```
字节SeedFold模型超越AlphaFold3，训练数据扩大147倍；北京AI产业两年内剑指万亿规模；豆包AI眼镜2000元内开售并获国家补贴；Stack Overflow问答量跌破18年前；腾讯ima一键生成PPT功能上线。
```

### AI前沿研究
1. **字节跳动**推出**SeedFold**分子结构预测模型，成功超越AlphaFold3。该模型采用宽度扩展和数据蒸馏技术，将配对表征隐藏维度从128提升至512，训练数据扩大147倍。在FoldBench多数任务中表现优于AlphaFold3，且线性版本推理效率更高。这一突破性进展有望加速药物研发和分子结构预测领域的发展。[相关链接](https://www.jiqizhixin.com/articles/2026-01-05-3)
2. **月之暗面**的**Kiwi-do**模型在AI大模型竞技场亮相，自称为Kimi系列模型。测试显示该模型在SVG绘图和VPCT视觉物理测试中表现优异，推测可能是AMA预告的K2-VL多模态版。模型训练数据截止2025年1月，表明月之暗面在多模态AI领域持续发力。[相关链接](https://mp.weixin.qq.com/s?__biz=MzIzNjc1NzUzMw==&mid=2247860058&idx=1&sn=148ce3ee32319a79ba0df4e85d243a1b)
![AI资讯：Kiwi-do模型SVG绘图作品展示](https://source.hubtoday.app/images/2026/01/news_01ke7cv6n4fg9azpkcasd0m70h.avif)
3. 研究者为深度特征提取器的信息传播速度推导出统一上界，覆盖欧几里得和非欧几里得域的神经网络模型。该发现在离散域输入中实现全局指数衰减，对CNN和散射网络均适用，为理解深度学习模型的内在机制提供了新视角。[相关链接](https://arxiv.org/abs/2502.15004)
4. 一支研究团队开发了跨层协同设计框架**SimPhony**，从器件物理到AI算法实现全覆盖。该工具支持快速评估和设计空间探索，通过AI驱动逆向设计实现可扩展的电子-光子AI(EPDA)栈，同时公开了光子边缘AI和Transformer推理架构，为硬件与算法协同设计提供了新工具。[相关链接](https://arxiv.org/abs/2601.00130)

### 行业展望与社会影响
1. **Stack Overflow**问答量跌破18年前水平，比2008年上线首月还少。AI Coding工具分走了大量流量，GitHub Copilot和ChatGPT改变了程序员习惯，过去遇到问题会发帖求助，现在直接询问AI。讽刺的是，平台自己接入AI后反而出现大量垃圾答案。[相关链接](https://www.qbitai.com/2026/01/366593.html)
![AI资讯：Stack Overflow问答数量下降趋势图](https://source.hubtoday.app/images/2026/01/news_01ke7cva7ye4js0j2c2nzcrpd6.avif)
2. **北京**发布九大行动计划，AI产业规模预计从4500亿元冲向万亿元。目前北京拥有约40家独角兽企业和近60家上市公司，豆包以1.72亿月活用户稳居榜首。具身智能和科学智能赛道正在加速布局，全产业链闭环正在成型。[相关链接](https://www.aibase.com/zh/news/24237)
3. **AI眼镜**首次纳入国家补贴范围，由工信部与财政部联合发文，可享受最高15%补贴，单台上限800元。华为、小米、Rokid等品牌首批入围，预计2026年出货量将突破200万台，政策红利加速AI眼镜从极客玩具向日常配件转变。[相关链接](https://www.aibase.com/zh/news/24239)
4. **月之暗面**完成C轮融资，到账5亿美元，由IDG领投，阿里、腾讯跟投。创始人杨植麟表示将激进扩显卡，加速K3训练，目标是让等效FLOPs提升一个数量级。公司营收策略聚焦Agent应用，而非单纯追求用户增长。[相关链接](https://mp.weixin.qq.com/s?__biz=MzIzNjc1NzUzMw==&mid=2247860058&idx=1&sn=148ce3ee32319a79ba0df4e85d243a1b)
